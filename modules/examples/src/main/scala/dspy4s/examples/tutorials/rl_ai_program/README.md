🚫 **Not ported.** DSPy has experimental reinforcement-learning optimizers that tune programs by treating
execution as an RL problem (the tutorials cover RL for privacy-conscious delegation and multi-hop research).
dspy4s has no RL optimizer, so these tutorials have no dspy4s counterpart.

For the optimization that is available today, see [`learn/optimization`](../../learn/optimization/README.md).
