# Optimize AI Programs with dspy4s

> Adapted for **dspy4s** from the DSPy docs page

DSPy's optimization tutorials (math reasoning, classification finetuning, advanced tool use, finetuning agents)
are mostly notebooks built on optimizers and finetuning that dspy4s hasn't ported yet.

What dspy4s offers today: few-shot demo optimization via `BootstrapFewShot` and
`BootstrapFewShotWithRandomSearch` — see [`learn/optimization/Optimizers.scala`](../../learn/optimization/Optimizers.scala)
and the [optimization overview](../../learn/optimization/README.md).

Not yet ported: MIPROv2, GEPA, COPRO, SIMBA, KNNFewShot, Ensemble, and any finetuning-based optimizer, so the
specific tutorials on this DSPy page don't have a dspy4s counterpart yet.
